메인 캐릭터 생성 게임

테이블 위의 모험을, 브라우저로.

캐릭터 시트를 만들고 엑셀로 저장/불러오기, 게임 화면에서 채팅과 주사위를 굴려 바로 진행할 수 있는 간단한 TRPG 웹페이지입니다.

캐릭터 생성으로 게임 화면으로
엑셀 연동
xlsx 내보내기/가져오기
✔
채팅
로그 + 시스템 메시지
✔
주사위
예: 3d6, 1d20+3
✔
기본 기능은 오프라인에서도 동작 (xlsx는 인터넷 필요)
추천 사용 흐름

1) 캐릭터 생성에서 시트 작성 → 2) xlsx로 저장 → 3) 게임 페이지에서 시트 불러오기 → 4) 채팅에 3d6 입력해 굴리기

시트 만들기 세션 시작
주의

이 프로젝트는 서버 없이 로컬 저장소(LocalStorage) 기반입니다. 다른 PC/브라우저로 옮기려면 xlsx 내보내기를 사용하세요. (xlsx 기능은 외부 라이브러리 CDN을 사용하므로 인터넷 연결이 필요할 수 있어요.)
